Ferox Developer Manual

Learn how to use and expand on an already kickass boilerplate

Key Features

SEO Friendly

Tightly integrated with next.js and SSR, making sure our apps are SEO friendly is as easy as pie

Integrated with WordPress

Clients need a place to manage their content, we got this covered thanks to the impressive WordPress REST API

Redux Integrating

Don't worry about the pains of setting up Redux, we got you covered. Including a basic store, redux, action and dispatches. Everything you need for Redux is taken care of.

Test Driven Development with Jest & Enzyme

Ensuring best practices are met, this boilerplate is already set up with Jest & Enzyme to ensure your app is bulletproof. We have also included some examples of working unit test to get you started.

Dynamic Layout Creation with ACF Pro

Whats cooler than being able to manage content for a React app in a CMS? Being able to create complex page layouts within your CMS and seeing them translate directly to your React app.

SASS Integration

Who wants to write in plain CSS? Not us, that is why we have set up production ready SASS compiling out of the box to take your styling to the next level
